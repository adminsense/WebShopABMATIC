# Owner-only git & publish

- **Never** run `git commit`, `git push`, amend, force-push, or `gh pr create` unless Marco explicitly asks in the current message.
- **Never** publish/deploy to Azure / `dotnet publish` for release unless he explicitly asks.
- Local `dotnet build` / `dotnet run` is allowed.
- When work is finished: summarize changes and **wait** — Marco commits and publishes himself.

Same rule: `AGENTS.md` · root process contract.
